# Xor App

Приложение для пакетной обработки файлов с применением операции XOR.

Поддерживает:

* поиск файлов по маске;
* обработку больших файлов чанками;
* паузу / продолжение / отмену;
* безопасную запись через временные файлы;
* автоматический перезапуск по таймеру;
* различные политики обработки дубликатов.
* XOR-обработка файлов 8-байтной маской;

## Использование

1. Выберите исходную папку;
2. Выберите папку результатов;
3. Укажите маску файлов;
4. Введите XOR-маску (16 hex символов);
5. Нажмите `Старт`.


## XOR-маска

Используется 8-байтная маска:

```text
0123456789ABCDEF
```

Поддерживаются HEX-символы:

* `0-9`
* `A-F`
* `a-f`
